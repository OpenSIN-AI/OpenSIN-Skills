# OpenSIN-Skills Boundaries

## Role
`OpenSIN-Skills` is the skill library and authoring surface for reusable agent expertise and operational actions.

## Hard rules

- Keep ownership claims scoped to skills and authoring standards.
- Do not let this repo redefine runtime, product, control-plane, or docs canon.
- Skill content may reference other owning repos, but should not replace them as source of truth.
