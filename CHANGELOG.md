# Changelog

All notable changes to OpenSIN-Skills will be documented in this file.

## [1.0.0] - 2026-04-18

### Added
- Initial release of OpenSIN-Skills
- 235 expertise/knowledge skills imported from claude-skills (by Alireza Rezvani, MIT)
- 47 operational/action skills ported from OpenSIN
- 9 personas (3 imported + 6 new OpenSIN personas)
- 8 orchestration patterns (4 imported + 4 new OpenSIN patterns)
- 13-pattern Skill Authoring Standard (10 upstream + 3 OpenSIN)
- validate-skill.py and catalog-generator.py (stdlib-only Python)
- OpenCode / Cursor / Aider / Windsurf integration support
- CONVENTIONS.md with OpenSIN coding standards
